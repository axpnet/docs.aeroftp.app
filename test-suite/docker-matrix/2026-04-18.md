# Docker Matrix - 2026-04-18

Matrice di integrazione su container Docker locali per FTP, SFTP (ed25519), WebDAV, S3/MinIO.

**Target**: `aeroftp-cli` (CLI) e logica provider condivisa
**Scope**: operazioni base (connect / ls / stat / get / put / mkdir / rm / mv / tree / hashsum), file da 10 MB e 100 MB con verifica SHA-256 end-to-end
**Profili usati**: Docker FTP, Docker SFTP ed25519, Docker WebDAV, Docker MinIO
