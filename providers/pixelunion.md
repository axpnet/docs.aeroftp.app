# PixelUnion

[PixelUnion](https://pixelunion.eu) is an EU-hosted managed Immich photo cloud based in the Netherlands. It offers 16 GB of free storage with full Immich functionality.

AeroFTP connects to PixelUnion using the same Immich REST API integration. PixelUnion is available as a preset in Discover Services for quick setup.

## Connection Settings

| Field | Value |
| --- | --- |
| Server URL | `https://yourname.pixelunion.eu` (your subdomain) |
| API Key | Generated in PixelUnion > User Settings > API Keys |

## How to Connect

1. Select **PixelUnion** from Discover Services.
2. Enter your subdomain URL (e.g. `https://aeroftp.pixelunion.eu`).
3. Paste your API key.
4. Click **Connect**.

## Features

All Immich features are supported. See the [Immich guide](/providers/immich) for the full list.

## Plans

| Plan | Storage | Price |
| --- | --- | --- |
| Free | 16 GB | Free |
| Paid plans | Up to 6 TB | See [pixelunion.eu](https://pixelunion.eu) |

## Tips

- Your subdomain is chosen during registration and cannot be changed later.
- PixelUnion data is stored in the EU (Netherlands), GDPR compliant.
- PixelUnion V.O.F. is registered in Alkmaar, Netherlands (KvK 97002348).
