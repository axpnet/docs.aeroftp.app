# Screenshots

A visual tour of AeroFTP's interface and features.

## Connection Screen

Connect to 25 protocols from a single interface. Saved servers with protocol badges, search, and one-click connect.

![Connection Screen](/screenshots/connection-screen-protocols.png)

## Multi-Tab File Manager

Browse multiple servers simultaneously with dual-panel layout, context menus, drag & drop, and file tags.

![Multi-Tab File Manager](/screenshots/multi-tab-file-manager.png)

## Sync & Transfer Queue

Bidirectional file synchronization with real-time transfer progress, speed graph, and activity log.

![Sync & Transfer](/screenshots/sync-transfer-queue.png)

## DevTools & AeroAgent

Integrated code editor (Monaco), SSH terminal, and AI assistant with 47 tools — all in resizable panels.

![DevTools & AeroAgent](/screenshots/devtools-aeroagent.png)

## Themes

Four built-in themes: Light, Dark, Tokyo Night, and Cyber. Consistent across all panels and modals.

![Settings Themes](/screenshots/settings-themes.png)
